зі»з»ҹи®ҫи®ЎпјҡеҰӮдҪ•и®ҫи®Ўй«ҳеҸҜз”Ёеҫ®жңҚеҠЎзі»з»ҹпјҲеҝ«йҖҹе®№зҒҫдёҺжөҒйҮҸеҲҮжҚўпјү

Posted on 2026-06-13 Edited on 2026-06-14 In System

й«ҳеҸҜз”Ёеҫ®жңҚеҠЎжҳҜжҜҸдёҖеңәзі»з»ҹи®ҫи®ЎйқўиҜ•зҡ„еҝ…иҖғйўҳгҖӮжң¬ж–Үд»ҘвҖқи®ҫи®ЎдёҖдёӘеҸҜд»Ҙеҝ«йҖҹе®№зҒҫзҡ„еҫ®жңҚеҠЎзі»з»ҹвҖқдёәдё»зәҝпјҢе®Ңж•ҙжўізҗҶйқўиҜ•е®ҳжңҹеҫ…еҗ¬еҲ°зҡ„зӯ”жЎҲжЎҶжһ¶пјҡеҰӮдҪ•жЈҖжөӢж•…йҡңгҖҒеҰӮдҪ•еҲҮжҚўжөҒйҮҸгҖҒеҰӮдҪ•жҒўеӨҚпјҢд»ҘеҸҠиғҢеҗҺзҡ„е…ій”®жқғиЎЎгҖӮ

1. йқўиҜ•йўҳиҝҳеҺҹдёҺз ҙйўҳ

йқўиҜ•е®ҳеёёи§Ғй—®жі•пјҡ

вҖңиҜ·и®ҫи®ЎдёҖдёӘеҫ®жңҚеҠЎзі»з»ҹпјҢиҰҒжұӮеҪ“жҹҗдёӘжңҚеҠЎжҲ–ж•ҙдёӘеҸҜз”ЁеҢәеҮәзҺ°ж•…йҡңж—¶пјҢиғҪеңЁ 30 з§’еҶ…е®ҢжҲҗжөҒйҮҸеҲҮжҚўпјҢе№¶дё”з”ЁжҲ·ж„ҹзҹҘзҡ„й”ҷиҜҜзҺҮжҺ§еҲ¶еңЁ 0.1% д»ҘдёӢгҖӮвҖқ

з ҙйўҳдёүжӯҘиө°пјҡ

  1. жҫ„жё…йңҖжұӮпјҡRTOпјҲжҒўеӨҚж—¶й—ҙзӣ®ж ҮпјүжҳҜеӨҡе°‘пјҹRPOпјҲжҒўеӨҚзӮ№зӣ®ж ҮпјүжҳҜеӨҡе°‘пјҹжҳҜеҚ•жңҚеҠЎж•…йҡңиҝҳжҳҜж•ҙдёӘ AZ/Region зә§ж•…йҡңпјҹ
  2. жӢҶи§Јеӯҗй—®йўҳпјҡж•…йҡңжЈҖжөӢ вҶ’ еҶізӯ– вҶ’ жөҒйҮҸеҲҮжҚў вҶ’ йӘҢиҜҒ вҶ’ еӣһж»ҡ
  3. еҲҶеұӮи®ҫи®ЎпјҡеҹәзЎҖи®ҫж–ҪеұӮпјҲDNS/LBпјүгҖҒжңҚеҠЎеұӮпјҲзҶ”ж–ӯ/йҷҗжөҒпјүгҖҒж•°жҚ®еұӮпјҲдё»д»Һ/еӨҡжҙ»пјү

2. ж•ҙдҪ“жһ¶жһ„еӣҫ

е…ій”®з»„д»¶пјҡGlobal Load BalancerпјҲжөҒйҮҸе…ҘеҸЈпјүгҖҒеҸҢ Region йғЁзҪІгҖҒи·Ё Region ж•°жҚ®еҗҢжӯҘгҖҒз»ҹдёҖзҡ„еҸҜи§ӮжөӢе№іеҸ°гҖӮ

3. ж•…йҡңжЈҖжөӢ

е®№зҒҫзҡ„з¬¬дёҖжӯҘжҳҜвҖқзҹҘйҒ“еқҸдәҶвҖқгҖӮжЈҖжөӢи¶Ҡеҝ«пјҢRTO и¶ҠдҪҺгҖӮ

3.1 еҲҶеұӮжЈҖжөӢдҪ“зі»

еұӮж¬Ў жЈҖжөӢжүӢж®ө е…ёеһӢе»¶иҝҹ
еҹәзЎҖи®ҫж–Ҫ Cloud Provider Health CheckгҖҒBGP и·Ҝз”ұж”¶ж•ӣ 1вҖ“5s
иҙҹиҪҪеқҮиЎЎ LB Active Health CheckпјҲHTTP 200 + е»¶иҝҹйҳҲеҖјпјү 2вҖ“10s
жңҚеҠЎзҪ‘ж ј Envoy/Istio Outlier DetectionпјҲиҝһз»ӯй”ҷиҜҜзҺҮпјү з§’зә§
дёҡеҠЎжҢҮж Ү жҲҗеҠҹзҺҮ / P99 / дёҡеҠЎиҪ¬еҢ–зҺҮ з§’-еҲҶй’ҹзә§
з»јеҗҲеҲӨж–ӯ е‘ҠиӯҰиҒҡеҗҲ + иҮӘеҠЁеҶізӯ–еј•ж“Һ ж•°з§’

3.2 Active Health Check vs Passive Detection

Active Health Checkпјҡ

  • LB жҲ– Sidecar е®ҡжңҹеҸ‘жҺўй’ҲиҜ·жұӮпјҲ/healthzпјүпјҢиҝһз»ӯ N ж¬ЎеӨұиҙҘеҲҷж‘ҳйҷӨ
  • дјҳзӮ№пјҡдё»еҠЁеҸ‘зҺ°гҖҒеҸҜзІҫзЎ®жҺ§еҲ¶йҳҲеҖј
  • зјәзӮ№пјҡжҺўй’ҲдёҚд»ЈиЎЁзңҹе®һжөҒйҮҸи·Ҝеҫ„пјҢеҸҜиғҪжјҸжҠҘвҖқеұҖйғЁж•…йҡңвҖқ

Passive DetectionпјҲOutlier Detectionпјүпјҡ

  • и§ӮжөӢзңҹе®һиҜ·жұӮзҡ„й”ҷиҜҜзҺҮ/иҝһжҺҘи¶…ж—¶пјҢеҠЁжҖҒй©ұйҖҗејӮеёёе®һдҫӢ
  • Envoy й…ҚзҪ®зӨәдҫӢпјҡ

жңҖдҪіе®һи·өпјҡдёӨиҖ…з»“еҗҲпјҢActive иҙҹиҙЈвҖқеҝ«йҖҹж‘ҳйҷӨжӯ»е®һдҫӢвҖқпјҢPassive иҙҹиҙЈвҖқиҜҶеҲ«жҠ–еҠЁе®һдҫӢвҖқгҖӮ

3.3 е…ій”® SLI жҢҮж Ү

йқўиҜ•ж—¶йңҖиҰҒиҜҙжё…жҘҡзӣ‘жҺ§д»Җд№Ҳпјҡ

е‘ҠиӯҰзӯ–з•Ҙе»әи®®з”Ё еӨҡзӘ—еҸЈе‘ҠиӯҰпјҲеҰӮеҗҢж—¶ж»Ўи¶і 5min зӘ—еҸЈе’Ң 1min зӘ—еҸЈпјүпјҢйҒҝе…Қзһ¬ж—¶жҠ–еҠЁи§ҰеҸ‘иҜҜжҠҘпјҡ

4. жөҒйҮҸеҲҮжҚў

жЈҖжөӢеҲ°ж•…йҡңеҗҺпјҢж ёеҝғй—®йўҳжҳҜвҖқжҖҺд№ҲеҲҮвҖқгҖӮйқўиҜ•е®ҳдјҡиҝҪй—®пјҡдјҡдёҚдјҡеҲҮеҮәж–°зҡ„ж•…йҡңпјҹ

4.1 жөҒйҮҸеҲҮжҚўеұӮж¬Ў

4.2 йҮ‘дёқйӣҖеҲҮжөҒзЁӢеәҸ

ж°ёиҝңдёҚиҰҒдёҖеҲҖеҲҮпјҢеҲҮжөҒеә”иҜҘжҳҜжёҗиҝӣејҸзҡ„пјҡ

е…ій”®еҲӨж–ӯпјҡеңЁжҜҸдёӘйҳ¶ж®өйғҪиҰҒжЈҖжҹҘзӣ®ж Ү Region жҳҜеҗҰиғҪжүҝеҸ—ж–°еўһжөҒйҮҸпјҲе®№йҮҸйў„з•ҷпјүгҖӮ

4.3 зҶ”ж–ӯеҷЁпјҲCircuit Breakerпјү

зҶ”ж–ӯеҷЁжҳҜжңҚеҠЎеұӮжңҖеёёз”Ёзҡ„иҮӘдҝқжүӢж®өпјҢдёүз§ҚзҠ¶жҖҒпјҡ

й…ҚзҪ®еҸӮж•°пјҲд»Ҙ Resilience4j дёәдҫӢпјүпјҡ

4.4 йҷҗжөҒдёҺйҷҚзә§

еҲҮжөҒиҝҮзЁӢдёӯпјҢзӣ®ж Ү Region дјҡжүҝжҺҘйўқеӨ–жөҒйҮҸпјҢеҝ…йЎ»жңүйҷҗжөҒе…ңеә•пјҡ

д»ӨзүҢжЎ¶ vs жјҸжЎ¶пјҡ

д»ӨзүҢжЎ¶пјҲToken Bucketпјү жјҸжЎ¶пјҲLeaky Bucketпјү
е…Ғи®ёзӘҒеҸ‘ вң… жҳҜ вқҢ еҗҰ
иҫ“еҮәйҖҹзҺҮ дёҚеқҮеҢҖ еҢҖйҖҹ
йҖӮз”ЁеңәжҷҜ API йҷҗжөҒпјҲе…Ғи®ёзҹӯж—¶зӘҒеҸ‘пјү еҗҺз«ҜдҝқжҠӨпјҲе№іж»‘жөҒйҮҸпјү

йҷҚзә§зӯ–з•ҘпјҲжҢүдјҳе…Ҳзә§пјүпјҡ

  1. иҝ”еӣһзј“еӯҳж•°жҚ®пјҲCache Fallbackпјү
  2. иҝ”еӣһй»ҳи®ӨеҖјпјҲDefault Responseпјү
  3. и°ғз”ЁйҷҚзә§жңҚеҠЎпјҲSimplified Serviceпјү
  4. зӣҙжҺҘжӢ’з»қе№¶иҝ”еӣһ 429пјҲжңҖеҗҺжүӢж®өпјү

5. ж•°жҚ®еұӮе®№зҒҫ

жөҒйҮҸеҲҮжҚўдәҶпјҢж•°жҚ®жҖҺд№ҲеҠһпјҹиҝҷжҳҜйқўиҜ•жңҖе®№жҳ“иў«иҝҪй—®зҡ„ең°ж–№гҖӮ

5.1 RPO vs RTO жқғиЎЎ

зӯ–з•Ҙ RTO RPO жҲҗжң¬
еҶ·еӨҮпјҲBackup & Restoreпјү е°Ҹж—¶зә§ ж•°е°Ҹж—¶ дҪҺ
жё©еӨҮпјҲWarm Standbyпјү еҲҶй’ҹзә§ з§’зә§ дёӯ
зғӯеӨҮпјҲActive-Passiveпјү з§’зә§ иҝ‘йӣ¶ й«ҳ
еӨҡжҙ»пјҲActive-Activeпјү жҜ«з§’зә§ йӣ¶ жһҒй«ҳ

5.2 дё»д»ҺеҲҮжҚўпјҲActive-Passiveпјү

йҖӮеҗҲеӨ§еӨҡж•°дә’иҒ”зҪ‘еңәжҷҜпјҢиҰҒзӮ№пјҡ

  1. ејӮжӯҘеӨҚеҲ¶пјҡдё»еә“еҶҷе…ҘеҗҺејӮжӯҘеҗҢжӯҘеҲ°д»Һеә“пјҢRPO < 1s еҸҜжҺҘеҸ—
  2. иҮӘеҠЁ FailoverпјҡдҪҝз”Ё MySQL MHA / PostgreSQL Patroni / Redis Sentinel еҒҡиҮӘеҠЁдё»д»ҺеҲҮжҚў
  3. йҳІжӯўи„‘иЈӮпјҡеј•е…ҘеӨ–йғЁд»ІиЈҒпјҲеҰӮ etcd / ZooKeeperпјүпјҢзЎ®дҝқеҗҢдёҖж—¶еҲ»еҸӘжңүдёҖдёӘдё»еә“

5.3 еӨҡжҙ»ж•°жҚ®еә“пјҲActive-Activeпјү

йҖӮеҗҲеҜ№ RTO/RPO жһҒиҮҙиҰҒжұӮзҡ„еңәжҷҜпјҲеҰӮж”Ҝд»ҳгҖҒдәӨжҳ“пјүпјҡ

  • еҲҶзүҮпјҲShardingпјүпјҡжҢүз”ЁжҲ· ID / Region еҲҶзүҮпјҢжҜҸдёӘ Region еҸӘеҶҷиҮӘе·ұзҡ„еҲҶзүҮпјҢйҒҝе…Қи·Ё Region еҶҷеҶІзӘҒ
  • еҶІзӘҒи§ЈеҶіпјҡLast-Write-WinsпјҲLWWпјүжҲ– CRDTпјҲConflict-Free Replicated Data Typesпјү
  • е…ЁеұҖдәӢеҠЎпјҡдҪҝз”ЁеҲҶеёғејҸдәӢеҠЎпјҲ2PC / SagaпјүпјҢдҪҶйңҖжҺҘеҸ—е»¶иҝҹеҚҮй«ҳ

6. иҮӘеҠЁеҢ–е®№зҒҫеҶізӯ–

6.1 е®№зҒҫзј–жҺ’жөҒзЁӢ

6.2 дәәе·Ҙе№Ійў„иҠӮзӮ№

иҮӘеҠЁеҢ–е®№зҒҫдёӯеҝ…йЎ»дҝқз•ҷдәәе·Ҙе№Ійў„зӮ№пјҡ

  • еҲҮжөҒеҲ° 100% д№ӢеүҚйңҖиҰҒдәәе·ҘзЎ®и®ӨпјҲйҳІжӯўиҮӘеҠЁеҢ–жҠҠеқҸжөҒйҮҸеёҰеҲ°еҒҘеә· Regionпјү
  • ж•°жҚ®еә“дё»д»ҺеҲҮжҚў е»әи®®еҚҠиҮӘеҠЁпјҲзі»з»ҹжЈҖжөӢ + дәәе·Ҙжү§иЎҢпјҢйҳІи„‘иЈӮпјү
  • еӣһж»ҡж“ҚдҪң еҝ…йЎ»йҡҸж—¶еҸҜжү§иЎҢпјҢдё”жңүжҳҺзЎ®зҡ„ Rollback Runbook

7. ж··жІҢе·ҘзЁӢйӘҢиҜҒ

е®№зҒҫж–№жЎҲи®ҫи®Ўе®ҢдёҚзӯүдәҺиғҪз”ЁпјҢйңҖиҰҒйҖҡиҝҮж··жІҢе·ҘзЁӢжҢҒз»ӯйӘҢиҜҒгҖӮ

7.1 еёёи§ҒжіЁе…Ҙж•…йҡңзұ»еһӢ

ж•…йҡңзұ»еһӢ жіЁе…Ҙж–№ејҸ йӘҢиҜҒзӣ®ж Ү
жңҚеҠЎе®һдҫӢе®•жңә kill -9 иҝӣзЁӢ Outlier Detection + жңҚеҠЎеҸ‘зҺ°
зҪ‘з»ңе»¶иҝҹ tc netem / toxiproxy и¶…ж—¶й…ҚзҪ®гҖҒзҶ”ж–ӯ
зҪ‘з»ңдёўеҢ… iptables drop йҮҚиҜ•зӯ–з•Ҙ
зЈҒзӣҳж»Ў fallocate еҶҷж»Ў й”ҷиҜҜеӨ„зҗҶгҖҒе‘ҠиӯҰ

Region ж•…йҡң е…Ё Region жөҒйҮҸеҪ’йӣ¶ и·Ё Region DR

7.2 GameDay жј”з»ғ

е®ҡжңҹпјҲжҜҸеӯЈеәҰпјүиҝӣиЎҢ GameDayпјҡ

  1. жҸҗеүҚйҖҡзҹҘзӣёе…іеӣўйҳҹпјҢзЎ®е®ҡжј”з»ғж—¶й—ҙзӘ—еҸЈ
  2. жіЁе…Ҙж•…йҡңпјҢи§ӮеҜҹзі»з»ҹиҮӘеҠЁе“Қеә”
  3. еҰӮи¶…иҝҮ RTO жңӘжҒўеӨҚпјҢдәәе·ҘжҺҘз®Ў
  4. жј”з»ғеҗҺеҶҷ Post-MortemпјҢжӣҙж–° Runbook

8. йқўиҜ•еҠ еҲҶйЎ№

йқўиҜ•ж—¶жҢү STAR з»“жһ„еӣһзӯ”пјҡ

  • SituationпјҡиҜҙжё…жҘҡи§„жЁЎпјҲQPSгҖҒиҠӮзӮ№ж•°гҖҒж•°жҚ®йҮҸпјү
  • TaskпјҡжҳҺзЎ® RTO/RPO зӣ®ж Ү
  • ActionпјҡйҖҗеұӮи®Іи§ЈжЈҖжөӢгҖҒеҲҮжөҒгҖҒж•°жҚ®е®№зҒҫ
  • ResultпјҡйҮҸеҢ–з»“жһңпјҲеҲҮжөҒж—¶й—ҙгҖҒй”ҷиҜҜзҺҮпјү

8.2 еёёи§ҒиҝҪй—®е’ҢеҸӮиҖғзӯ”жЎҲ

Q: еҰӮдҪ•йҳІжӯўеҲҮжөҒж—¶йҖ жҲҗж–°зҡ„ж•…йҡңпјҲеҲҮжөҒйӣӘеҙ©пјүпјҹ

A: еҲҮжөҒеүҚжЈҖжҹҘзӣ®ж Ү Region е®№йҮҸпјҢе…Ҳжү©е®№еҶҚеҲҮжөҒпјӣеҲҮжөҒиҝҮзЁӢдёӯжҢҒз»ӯзӣ‘жҺ§зӣ®ж Ү Region зҡ„ CPU/еҶ…еӯҳ/йҳҹеҲ—ж·ұеәҰпјӣи®ҫзҪ®жңҖеӨ§еҲҮжөҒйҖҹеәҰпјҢе…Ғи®ёйҡҸж—¶ RollbackгҖӮ

Q: DNS еҲҮжҚўеҫҲж…ўпјҢжҖҺд№ҲеҠ йҖҹпјҹ

A: жҸҗеүҚе°Ҷ TTL йҷҚеҲ° 30вҖ“60sпјҲжӯЈеёёж—¶жҳҜ 300sпјүпјӣдҪҝз”Ё Anycast + BGP и·Ҝз”ұи°ғж•ҙд»Јжӣҝ DNSпјҢз§’зә§з”ҹж•ҲпјӣеңЁе®ўжҲ·з«Ҝ SDK дёӯз»•иҝҮ DNS зј“еӯҳпјҢзӣҙжҺҘеҲ·ж–° IP еҲ—иЎЁгҖӮ

Q: еӨҡжҙ»жһ¶жһ„дёӢеҰӮдҪ•и§ЈеҶіж•°жҚ®дёҖиҮҙжҖ§пјҹ

A: жҢүз…§ж•°жҚ®зҡ„еҶҷе…ҘжқҘжәҗеҲҶзүҮпјҢйҒҝе…ҚеҗҢдёҖж•°жҚ®еңЁеӨҡдёӘ Region еҗҢж—¶еҶҷе…ҘпјӣеҜ№дәҺеҝ…йЎ»и·Ё Region зҡ„еҶҷпјҲеҰӮе…ЁеұҖеә“еӯҳпјүпјҢдҪҝз”ЁеҲҶеёғејҸдәӢеҠЎпјҲSaga иЎҘеҒҝжЁЎејҸпјүпјҢе№¶жҺҘеҸ—жңҖз»ҲдёҖиҮҙжҖ§пјӣеҜ№ејәдёҖиҮҙеңәжҷҜпјҲж”Ҝд»ҳпјүпјҢеҸӘеңЁдё» Region еҶҷпјҢеӨҡжҙ»йҖҖеҢ–дёәиҜ»еӨҡжҙ»гҖӮ

Q: еҰӮдҪ•еҢәеҲҶвҖқеұҖйғЁжҠ–еҠЁвҖқе’ҢвҖқзңҹе®һж•…йҡңвҖқпјҢйҒҝе…ҚиҜҜеҲҮжөҒпјҹ

A: дҪҝз”ЁеӨҡзӘ—еҸЈе‘ҠиӯҰпјҲзҹӯзӘ—еҸЈи§ҰеҸ‘йў„иӯҰпјҢй•ҝзӘ—еҸЈи§ҰеҸ‘еҲҮжөҒпјүпјӣеј•е…ҘеӨҡдёӘзӢ¬з«Ӣзҡ„жҺўжөӢжәҗпјҲLB еҒҘеә·жЈҖжҹҘ + дёҡеҠЎжҲҗеҠҹзҺҮ + еӨ–йғЁжҺўжөӢпјүпјҢиҮіе°‘дёӨдёӘжқҘжәҗеҗҢж—¶ејӮеёёжүҚи§ҰеҸ‘еҲҮжөҒпјӣи®ҫзҪ®жңҖе°Ҹи§ӮжөӢж—¶й—ҙпјҲеҰӮиҝһз»ӯ 10s ејӮеёёпјүгҖӮ

8.3 и®ҫи®ЎжқғиЎЎжҖ»з»“

еҶізӯ–зӮ№ йҖүйЎ№ A йҖүйЎ№ B жҺЁиҚҗ
жЈҖжөӢйҖҹеәҰ жҝҖиҝӣйҳҲеҖјпјҲеҝ«дҪҶиҜҜжҠҘеӨҡпјү дҝқе®ҲйҳҲеҖјпјҲж…ўдҪҶеҮҶзЎ®пјү еӨҡзӘ—еҸЈз»“еҗҲ
еҲҮжөҒйҖҹеәҰ з«ӢеҚіе…ЁеҲҮпјҲеҝ«дҪҶйЈҺйҷ©еӨ§пјү жёҗиҝӣеҲҮжөҒпјҲж…ўдҪҶе®үе…Ёпјү жёҗиҝӣеҲҮжөҒ
ж•°жҚ®дёҖиҮҙ ејәдёҖиҮҙпјҲе»¶иҝҹй«ҳпјү жңҖз»ҲдёҖиҮҙпјҲе»¶иҝҹдҪҺпјү жҢүдёҡеҠЎйҖүжӢ©
иҮӘеҠЁеҢ–зЁӢеәҰ е…ЁиҮӘеҠЁпјҲеҝ«дҪҶйЈҺйҷ©й«ҳпјү е…ЁжүӢеҠЁпјҲе®үе…ЁдҪҶж…ўпјү еҚҠиҮӘеҠЁ+дәәе·ҘзЎ®и®Өе…ій”®жӯҘйӘӨ

жҖ»з»“

и®ҫи®Ўеҝ«йҖҹе®№зҒҫзҡ„еҫ®жңҚеҠЎзі»з»ҹпјҢж ёеҝғжҳҜеҝ«йҖҹжЈҖжөӢ + е®үе…ЁеҲҮжөҒ + ж•°жҚ®дҝқйҡңдёүдҪҚдёҖдҪ“пјҡ

  1. жЈҖжөӢпјҡеҲҶеұӮжЈҖжөӢпјҲLB жҺўй’Ҳ + Outlier Detection + дёҡеҠЎжҢҮж ҮпјүпјҢеӨҡзӘ—еҸЈе‘ҠиӯҰеҮҸе°‘иҜҜжҠҘ
  2. еҲҮжөҒпјҡGlobal LB + жңҚеҠЎзҪ‘ж је®һзҺ°з§’зә§еҲҮжөҒпјҢжёҗиҝӣејҸпјҲ10%вҶ’50%вҶ’100%пјүйҒҝе…ҚйӣӘеҙ©пјҢй…ҚеҗҲзҶ”ж–ӯйҷҗжөҒеҒҡиҮӘдҝқ
  3. ж•°жҚ®пјҡж №жҚ® RTO/RPO йҖүжӢ©дё»д»ҺзғӯеӨҮжҲ–еӨҡжҙ»пјҢзј“еӯҳжҸҗеүҚйў„зғӯпјҢйҳІжӯўеҲҮжөҒеҗҺзј“еӯҳеҮ»з©ҝ
  4. йӘҢиҜҒпјҡж··жІҢе·ҘзЁӢ + е®ҡжңҹ GameDayпјҢи®©е®№зҒҫиғҪеҠӣжҢҒз»ӯеҸҜдҝЎ

йқўиҜ•ж ёеҝғпјҡдёҚиҰҒеҸӘиҜҙвҖқжҲ‘дјҡз”Ё Kubernetes йғЁзҪІеӨҡеүҜжң¬вҖқпјҢиҖҢжҳҜиҰҒеұ•зӨәдҪ еҜ№ж•…йҡңжЈҖжөӢе»¶иҝҹгҖҒеҲҮжөҒе®үе…ЁжҖ§гҖҒж•°жҚ®дёҖиҮҙжҖ§жқғиЎЎзҡ„ж·ұе…ҘжҖқиҖғгҖӮ
